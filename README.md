# 🚗 BMW Interactive Showroom

An interactive and visually engaging BMW model showcase built using **HTML, CSS, and JavaScript**. This project delivers a modern, responsive, and game-like user experience with smooth animations, dynamic UI interactions, and real-time features.

---

## ✨ Features

* 🎯 **Interactive Model Cards** – Click any BMW model to view details in a modal
* 🎮 **Game-Style Animations** – Smooth transitions, hover tilt effects, and click feedback
* 🔴 **Special M Competition Effects** – Unique high-impact animations for M-series models
* 🔍 **Search Functionality** – Filter models in real-time
* 📱 **Responsive Design** – Works seamlessly across devices
* 📩 **Form Validation** – Contact form with input validation using JavaScript
* 🌌 **Parallax Scrolling** – Dynamic background movement
* ✨ **Custom Cursor & Effects** – Enhanced UI with glow and ripple effects

---

## 🛠️ Technologies Used

* **HTML5** – Structure and layout
* **CSS3** – Styling, responsiveness, and animations
* **JavaScript (Vanilla JS)** – DOM manipulation, interactivity, and logic
